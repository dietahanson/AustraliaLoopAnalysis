Description

Example R code for analyses presented in the main text is provided in simulation.r, bayestest.r, interface.r, mutual.r, tree.r, distance.r. Comments are provided at the beginning of each of these files describing the application of network analysis algorithms, and comments throughout annotate the steps involved. Dependencies are community.r, dia.r, and tk.r. These analyses require signed directed graphs created in Dia (available from http://live.gnome.org/Dia) as inputs. Example models provided as part of this supplement are macquarie.dia, mesocosm1.dia, mesocosm2.dia, and fivevariable1.dia–fivevariable5.dia.
