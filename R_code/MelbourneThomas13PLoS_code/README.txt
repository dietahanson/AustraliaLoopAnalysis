
R code for analyses presented in the main text is provided in the following files: preylimitation.r, wapextended.r and wapspatial.r. Dependencies are community.r and dia.r. These analyses require signed directed graphs created in Dia (available from http://live.gnome.org/Dia) as inputs. The three models provided are preylimitation.dia, wapextended.dia and wapspatial.dia.
